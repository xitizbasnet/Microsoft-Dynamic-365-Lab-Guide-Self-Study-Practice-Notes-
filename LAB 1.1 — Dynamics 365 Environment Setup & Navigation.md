# LAB 1.1 — Dynamics 365 Environment Setup & Navigation

> **Lab Objective**
>
> Access your Dynamics 365 trial environment, explore the available applications, and understand the navigation structure.

---

# Learning Objectives

After completing this lab, you will be able to:

* Access a Microsoft Dynamics 365 trial environment.
* Navigate between Dynamics 365 applications.
* Explore entities, records, and system views.
* Configure personal user settings.
* Apply recommended Dynamics 365 best practices.

---

# Prerequisites

Before starting this lab, ensure you have:

* A Microsoft 365 work or school account.
* Internet access.
* A supported modern web browser (Microsoft Edge or Google Chrome recommended).

---

# Lab Steps

## Step 1 — Sign Up for a Dynamics 365 Trial

**Objective:** Create and access a Dynamics 365 trial environment.

### Procedure

1. Go to:
   `https://dynamics.microsoft.com/en-us/dynamics-365-free-trial/`

2. Select **Customer Service** or **Sales** trial.

3. Click **Start free trial**.

4. Sign in with your Microsoft 365 work/school account.

5. Complete the setup wizard by providing:

   * Organization name
   * Region
   * Currency

> 💡 **Note**
>
> The trial environment may take several minutes to provision before it becomes available.

---

## Step 2 — Navigate the App Launcher

**Objective:** Become familiar with the Dynamics 365 application launcher and primary navigation.

### Procedure

1. Click the **9-dot App Launcher (waffle icon)** located in the upper-left corner.

2. View all installed Dynamics 365 applications.

3. Open **Customer Service Hub**.

4. Observe the **Sitemap** located on the left navigation pane.

5. Explore the following areas:

* 📊 Dashboards
* 📅 Activities
* 📁 Cases
* 📥 Queues
* 📚 Knowledge Base

> 💡 **Tip**
>
> Spend a few minutes exploring each area to understand how users navigate between modules.

---

## Step 3 — Explore Entities & Views

**Objective:** Learn how records are organized and displayed.

### Procedure

1. Open **Cases**.

2. Switch between the available views:

* Active Cases
* My Active Cases
* Resolved Cases

3. Open any record.

4. Observe the following components:

* 🕒 Timeline
* 🔗 Related tab
* 📈 Business Process Flow (top bar)

5. Use **Advanced Find (Ctrl + F)** to search across entities.

> **Note**
>
> Different entities may display different forms and related information depending on configuration.

---

## Step 4 — Customize Personal Settings

**Objective:** Configure your individual user preferences.

### Procedure

1. Select the **Gear** icon.

2. Open **Personalization Settings**.

3. Configure the following settings:

* Time Zone → **Gulf Standard Time (UAE)**
* Default home page
* Records per page
* Email synchronization preferences

> 💡 **Tip**
>
> Personal settings affect only your user experience and do not change system-wide configuration.

# ✅ Best Practice Tips

> **Important**

✔ Always use a dedicated **trial organization** for labs—never the production environment.

✔ Learn the following keyboard shortcuts:

| Shortcut     | Function                |
| ------------ | ----------------------- |
| **Ctrl + N** | Create a new record     |
| **Ctrl + S** | Save the current record |
| **Alt + F4** | Close the current form  |

✔ Bookmark the Dynamics 365 Admin Center:

`admin.powerplatform.microsoft.com`

✔ Use the **In-App Help (?)** panel to find documentation without leaving Dynamics 365.
